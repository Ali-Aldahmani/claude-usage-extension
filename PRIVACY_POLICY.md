# Privacy Policy

Claude Usage Tracker Chrome Extension respects your privacy.

## Data Collection

This extension does not collect, store, or transmit any personal data to external servers.

## Cookies

The extension reads the `sessionKey` cookie from `claude.ai` only to authenticate requests to the Claude usage API. This cookie is accessed locally and is never stored or sent to third-party services.

## Local Storage

User preferences such as refresh interval, notification settings, and profile configurations are stored locally using Chrome's storage API. This data never leaves the user's device.

## Notifications

Notifications are used only to alert the user when their Claude usage approaches certain limits.

## Data Sharing

No data is sold, shared, or transmitted to third parties.

## External Requests

The extension communicates only with the official Claude service at:

https://claude.ai

These requests are used solely to retrieve usage statistics.

## Changes

This privacy policy may be updated if the extension features change.
